/score/create/

Mesh creation commands.

Commands

boxMesh
cylinderMesh
realWorldLogVol
probe

boxMesh [MeshName]

Create scoring box mesh.

Available at all Geant4 states.

Parameters
MeshName type s

cylinderMesh [MeshName]

Create scoring mesh.

Available at all Geant4 states.

Parameters
MeshName type s

realWorldLogVol [logVol] [copyNumberLevel]

Define scorers to a logical volume defined in the real world.
- Name of the specified logical volume is used as the mesh name.
- /score/mesh commands do not affect for this mesh.
- If copyNumberLevel is set, the copy number of that-level higher
in the geometrical hierarchy is used as the index.

Available at all Geant4 states.

Parameters
logVol type s
copyNumberLevel type i Omittable : default value = 0 Parameter range : copyNumberLevel>=0

probe [pname] [halfSize] [unit] [checkOverlap]

Define scoring probe.
halfSize defines the half-width of the probing cube.

Available at all Geant4 states.

Parameters
pname type s
halfSize type d
unit type s Omittable : default value = mm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi
checkOverlap type b Omittable : default value = 0
