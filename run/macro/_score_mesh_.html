boxSize [Di] [Dj] [Dk] [Unit]

Define size of the scoring mesh.
Dx Dy Dz unit

Range : Di>0. && Dj>0. && Dk>0.

Available at all Geant4 states.

Parameters
Di type d
Dj type d
Dk type d
Unit type s Omittable : default value = mm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

cylinderSize [R] [Dz] [unit]

R Dz unit

Available at all Geant4 states.

Parameters
R type d Parameter range : R>0
Dz type d Parameter range : Dz>0
unit type s Omittable : default value = mm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

cylinderRMin [RMin] [Unit]

Define the inner radius of the tube mesh.
This command is not needed for cylinder mesh

Range : RMin>=0.

Available at all Geant4 states.

Parameters
RMin type d
Unit type s Omittable : default value = mm Parameter candidates : pc km m cm mm um nm Ang fm parsec kilometer meter centimeter millimeter micrometer nanometer angstrom fermi

cylinderAngles [startPhi] [deltaPhi] [unit]

Define starting angle and span for tube segment mesh.
This command is not needed for cylinder mesh

Available at all Geant4 states.

Parameters
startPhi type d
deltaPhi type d Parameter range : deltaPhi>0.
unit type s Omittable : default value = deg Parameter candidates : rad mrad deg radian milliradian degree

nBin [Ni] [Nj] [Nk]

Define segments of the scoring mesh.
[usage] /score/mesh/nBin
In case of boxMesh, parameters are given in
Ni :(int) Number of bins i (in x-axis)
Nj :(int) Number of bins j (in y-axis)
Nk :(int) Number of bins k (in z-axis)
In case of cylinderMesh, parameters are given in
Nr :(int) Number of bins in radial axis
Nz :(int) Number of bins in z axis
Nphi:(int) Number of bins in phi axis

Available at all Geant4 states.

Parameters
Ni type i Parameter range : Ni>0
Nj type i Parameter range : Nj>0
Nk type i Parameter range : Nk>0
