kineticEnergy [fname] [elow] [ehigh] [unit]

Kinetic energy filter.
[usage] /score/filter/kineticEnergy fname Elow Ehigh unit
fname :(String) Filter Name
Elow :(Double) Lower edge of kinetic energy
Ehigh :(Double) Higher edge of kinetic energy
unit :(String) unit of given kinetic energy

Available at all Geant4 states.

Parameters
fname type s
elow type d Omittable : default value = 0.0
ehigh type d Omittable : default value = 1.79769e+308
unit type s Omittable : default value = keV Parameter candidates : eV keV MeV GeV TeV PeV meV J electronvolt kiloelectronvolt megaelectronvolt gigaelectronvolt teraelectronvolt petaelectronvolt millielectronVolt joule
