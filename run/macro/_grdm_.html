Commands

setTimeWindow

setTimeWindow [time window] [Unit]

Time window to split a radioactive decay chain.
If a daughter nucleus appears after the window,
it is saved in a new entry in the output ntuple.
---Set it to <=0 to disable the splitting---

Available Geant4 state(s) : PreInit Idle

Parameters
time window type d
Unit type s Omittable : default value = s Parameter candidates : s ms us ns ps min h d y second millisecond microsecond nanosecond picosecond minute hour day year
