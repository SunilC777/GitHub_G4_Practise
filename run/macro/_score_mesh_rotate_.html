/score/mesh/rotate/

Mesh rotation commands.

Commands

rotateX
rotateY
rotateZ

rotateX [Rx] [Unit]

Rotate the scoring mesh in X axis.

Available at all Geant4 states.

Parameters
Rx type d
Unit type s Omittable : default value = deg Parameter candidates : rad mrad deg radian milliradian degree

Available at all Geant4 states.
