arrow3DLineSegmentsPerCircle [number]

Defines number of line segments per circle for drawing 3D arrows for future "/vis/scene/add/" commands.

Range : number >= 3

Available at all Geant4 states.

Parameters
number type i Omittable : default value = 6

colour [red] [green] [blue] [alpha]

Defines colour and opacity for future "/vis/scene/add/" commands.
(Except "/vis/scene/add/text" commands - see "/vis/set/textColour".)
Accepts (a) RGB triplet. e.g., ".3 .4 .5", or (b) string such as "white", "black", "grey", "red"...or (c) an additional number for opacity, e.g., ".3 .4 .5 .6" or "grey ! ! .6" (note "!"'s for unused parameters).
Default: white and opaque.

Available at all Geant4 states.

Parameters
red type s Omittable : default value = 1.
green type d Omittable : default value = 1
blue type d Omittable : default value = 1
alpha type d Omittable : default value = 1

extentForField [xmin] [xmax] [ymin] [ymax] [zmin] [zmax] [unit]

Sets an extent for future "/vis/scene/add/*Field" commands.
The default is a null extent, which is interpreted by the commands as the extent of the whole scene.

Available at all Geant4 states.

Parameters
xmin type d
xmax type d
ymin type d
ymax type d
zmin type d
zmax type d
unit type s

lineWidth [lineWidth]

Defines line width for future "/vis/scene/add/" commands. See "/vis/viewer/set/lineWidth" for more information.

Range : lineWidth >= 1.

Available at all Geant4 states.

Parameters
lineWidth type d Omittable : default value = 1

textColour [red] [green] [blue] [alpha]

Defines colour and opacity for future "/vis/scene/add/text" commands.
Accepts (a) RGB triplet. e.g., ".3 .4 .5", or (b) string such as "white", "black", "grey", "red"...or (c) an additional number for opacity, e.g., ".3 .4 .5 .6" or "grey ! ! .6" (note "!"'s for unused parameters).
Default: blue and opaque.

Available at all Geant4 states.

Parameters
red type s Omittable : default value = 0.
green type d Omittable : default value = 0
blue type d Omittable : default value = 1
alpha type d Omittable : default value = 1

textLayout [layout]

Defines layout future "/vis/scene/add/text" commands.
"left" (default) for left justification to provided coordinate.
"centre" or "center" for text centered on provided coordinate.
"right" for right justification to provided coordinate.
Default: left.

Available at all Geant4 states.

Parameters
layout type s Omittable : default value = left Parameter candidates : left centre center right

textSize [textSize]

Defines text size (pixels) for future "/vis/scene/add/" commands.

Range : textSize >= 8.

Available at all Geant4 states.

Parameters
textSize type d Omittable : default value = 12
