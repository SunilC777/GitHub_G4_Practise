Analysis batch plotting control

Commands

setStyle [Style]

Only one plotting style is available in low resolution: inlib_default: PAW style with low resolution fonts

Available Geant4 state(s) : PreInit Idle

Parameters
Style type s Parameter candidates : inlib_default

setLayout [columns] [rows]

Set page layout (number of columns and rows per page). Supported layouts: columns = 1 .. maxValueAllowed rows = 1 .. maxValueAllowed, and >= columns"

Available Geant4 state(s) : PreInit Idle

Parameters
columns type i Parameter range : columns>=1 && columns<=2
rows type i Parameter range : rows>=1 && rows<=3

setDimensions [width] [height]

Set the plotter window size (width and height) in pixels.

Available Geant4 state(s) : PreInit Idle

Parameters
width type i
height type i
